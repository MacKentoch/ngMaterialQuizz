ngMaterialQuizz
=====

##What is it?

An angular material quizz (ES6).

##try it

###fast way

[link here](https://rawgit.com/MacKentoch/ngMaterialQuizz/master/public/index.html)

###on your own

install npm packages
```bash
npm install
```
then jspm ones
```bash
jspm install
```


then launch server with :
```bash
gulp connect
```


*Note :*
- install jspm if you don't have it already
```bash
npm install jspm -g
